Support

Help for install issues, sync problems, widgets, Apple Watch, and support questions.

Use this page when AshTag is not behaving as expected, when Cloud Sync looks wrong between devices, or when you need help with exports, permissions, widgets, or the Apple Watch companion.

Common help topics

  • Install or update issues
  • Cloud Sync across iPhone and iPad
  • Import and export questions
  • Health heart-rate attachment behavior
  • Location permission or hotspot-map behavior
  • Widget availability or refresh behavior
  • Apple Watch syncing and complication refresh
  • Bug reports or general support questions

What to include in a report

  • Device model
  • iOS version and watchOS version, if relevant
  • AshTag version and build number
  • What you expected
  • What happened instead

If Cloud Sync is involved, include screenshots of the Today screen’s Sync Status card on each device if possible.

Current public support channel

AshTag’s current public support intake is a structured GitHub support form hosted in the public ashtag-site repository.

Use it for install issues, Cloud Sync problems, widget or Apple Watch bugs, and other support questions. A GitHub account is currently required to submit the form.

If this is urgent
