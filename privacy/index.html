Privacy Policy

AshTag stays device-first unless the user chooses more.

AshTag is a smoking-cessation companion focused on private, device-first logging with optional sync and optional health context when the user explicitly turns those features on.

What AshTag stores

  • Cigarette logs
  • Craving logs
  • Rescue actions and reflection entries
  • Support contacts the user chooses to save
  • Optional place labels and coordinates when location is enabled
  • Optional heart-rate samples and source details when Health access is enabled
  • Optional step counts during Take a Walk when Motion & Fitness support is enabled

How permissions work

  • Location is optional and supports hotspot insights plus place-tagged history.
  • Notifications are optional and support reminders and check-ins.
  • Health access is optional and only reads recent heart-rate context the user chooses to attach.
  • Motion & Fitness is optional and only supports the walk rescue flow when enabled.

Data handling posture

  • No account is required.
  • History stays on the device by default.
  • Cloud Sync is optional and uses signed Apple devices on the same iCloud account.
  • Users can export, import, delete, and reset data from inside the app.

Need help with privacy or sync?

If you have questions about what AshTag stores, how Cloud Sync behaves, or what permissions do in the current release, use the public support page.
